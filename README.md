<h1>Notícias Cidade</h1>

Este projeto é um site voltado para o ramo de notícias diárias de uma cidade, ele apresenta várias funcionalidades e notícias diversas.

<h1>Tecnologias Utilizadas</h1>

HTML<br>
CSS3
